# Markdown 转 Word 完整格式规范

## 📋 格式总览

| 元素 | Markdown 语法 | 字体 | 字号 | 颜色 | 样式 | 对齐 | 首行缩进 | 行距 | 间距 |
|------|--------------|------|------|------|------|------|----------|------|------|
| **文档标题** | `# 标题` | 方正小标宋简体 | 22pt | 黑色 | 加粗 | 居中 | 无 | 1.0 | 段后12pt |
| **一级标题** | `# 标题` | 方正小标宋简体 | 22pt | 黑色 | 加粗 | 居中 | 无 | 1.5 | 段前180pt 段后60pt |
| **二级标题** | `## 标题` | 楷体 | 16pt | 黑色 | 加粗 | 左对齐 | 无 | 1.5 | 段前120pt 段后40pt |
| **三级标题** | `### 标题` | 仿宋 | 16pt | 黑色 | 加粗 | 左对齐 | 无 | 1.5 | 段前100pt 段后30pt |
| **正文** | 普通文本 | 仿宋 | 12pt | 黑色 | 正常 | 左对齐 | **2字符** | 1.5 | 正常 |
| **列表** | `* 项目` | 仿宋 | 12pt | 黑色 | 正常 | 左对齐 | **2字符** | 1.5 | 正常 |
| **引用** | `> 内容` | 仿宋 | 11pt | 黑色 | 斜体 | 左对齐 | **2字符** | 1.5 | 正常 |

---

## 🎯 详细说明

### 1. 文档标题（封面大标题）

```markdown
# 汽车工程学院2025年度辅导员个人工作总结
```

**格式**:
- 字体：方正小标宋简体
- 字号：22pt（二号）
- 颜色：黑色 RGB(0,0,0)
- 样式：加粗
- 对齐：居中
- **下方无横线/边框** ✅
- 行距：1.0倍
- 段后间距：12pt

---

### 2. 一级标题

```markdown
# 一、 强化思想引领，筑牢信仰之基
```

**格式**:
- 字体：方正小标宋简体
- 字号：22pt（二号）
- 颜色：黑色
- 样式：加粗
- 对齐：居中
- 首行缩进：无
- 行距：1.5倍
- 段前间距：180pt
- 段后间距：60pt

**注意**: 内容中的第一个一级标题会被跳过（已作为文档标题）

---

### 3. 二级标题

```markdown
## 二、 聚焦学业发展，强化学风建设
```

**格式**:
- 字体：楷体
- 字号：16pt（三号）
- 颜色：黑色
- 样式：加粗
- 对齐：左对齐
- 首行缩进：无
- 行距：1.5倍
- 段前间距：120pt
- 段后间距：40pt

---

### 4. 三级标题

```markdown
### 三级标题内容
```

**格式**:
- 字体：仿宋
- 字号：16pt（三号）
- 颜色：黑色
- 样式：加粗
- 对齐：左对齐
- 首行缩进：无
- 行距：1.5倍
- 段前间距：100pt
- 段后间距：30pt

---

### 5. 正文段落

```markdown
根据学校及学院年度考核工作安排...
```

**格式**:
- 字体：仿宋
- 字号：12pt（小四）
- 颜色：黑色
- 样式：正常
- 对齐：左对齐
- **首行缩进：2字符（0.3英寸）** ✅
- 行距：1.5倍

**支持行内格式**:
- `**加粗文本**` → 加粗
- `*斜体文本*` → 斜体

---

### 6. 列表

```markdown
*   **主题班会系统深入：** 全年围绕不同主题精心组织...
*   **个别指导精准到位：** 针对学生的思想动态...
```

**格式**:
- 字体：仿宋
- 字号：12pt（小四）
- 颜色：黑色
- 样式：正常
- 对齐：左对齐
- **首行缩进：2字符** ✅
- 行距：1.5倍
- 列表符号：`•`（程序自动统一）

**注意**:
- 支持多种列表符号：`*`, `-`, `•`, `+`
- 列表符号后的多余空格会被自动移除
- 支持嵌套列表（通过缩进级别）

---

### 7. 引用块

```markdown
> **说明：** 本总结所述内容均以实际工作记录为依据。
```

**格式**:
- 字体：仿宋
- 字号：11pt
- 颜色：黑色
- 样式：斜体
- 对齐：左对齐
- **首行缩进：2字符** ✅
- 行距：1.5倍

---

### 8. 分隔线

```markdown
---
```

**格式**:
- 显示为：`_` 重复50次
- 不使用首行缩进

## ✅ 特殊功能

### 1. 标题自动提取

- 从第一个一级标题提取文档标题
- 自动用作文件名和封面大标题
- 内容中的第一个一级标题会被跳过（避免重复）

### 2. 横线自动移除

- 文档标题下方的横线已移除
- 符合中文公文格式要求

### 3. 颜色统一

- 所有文字强制设置为黑色 RGB(0,0,0)
- 确保打印效果一致

### 4. 列表格式优化

- 自动统一列表符号为 `•`
- 自动移除多余空格
- Word 自动处理符号与文字的间距

## 📝 Markdown 文件要求

### 1. 编码
- ✅ 必须使用 **UTF-8** 编码
- ❌ 不能使用 GBK、GB2312 等其他编码

### 2. 引号使用
- ✅ 推荐：直角引号 `「」`
- ⚠️ 避免：中文双引号 `""`（可能导致解析错误）

### 3. 标题层级
- ✅ 一级标题：`# 标题`
- ✅ 二级标题：`## 标题`
- ✅ 三级标题：`### 标题`
- ⚠️ 不建议使用四级以上标题

### 4. 文件命名

**标题中的特殊字符**（会被自动移除）:
```
< > : " / \ | ? *
```

## 🎨 公文规范参考

本工具遵循以下标准：

1. **《党政机关公文格式》（GB/T 9704-2012）**
   - 正文首行缩进2字符
   - 标题使用方正小标宋简体
   - 正文使用仿宋

2. **《标点符号用法》（GB/T 15834-2011）**
   - 正确使用引号、冒号等标点

3. **中文排版规范**
   - 行距1.5倍
   - 段落间距适当
   - 标题居中或左对齐
