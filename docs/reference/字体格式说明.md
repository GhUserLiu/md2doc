# 字体格式说明

## 各级标题和正文字体规范

| 元素 | Markdown 语法 | 字体 | 字号 | 样式 | 对齐方式 |
|------|--------------|------|------|------|----------|
| **文档标题** | `# 标题` | 方正小标宋简体 | 22pt (二号) | 加粗 | 居中 |
| **一级标题** | `# 标题` | 方正小标宋简体 | 22pt (二号) | 加粗 | 居中 |
| **二级标题** | `## 标题` | 楷体 | 16pt (三号) | 加粗 | 左对齐 |
| **三级标题** | `### 标题` | 仿宋 | 16pt (三号) | 加粗 | 左对齐 |
| **正文** | 普通文本 | 仿宋 | 12pt (小四) | 正常 | 左对齐 |
| **列表** | `* 项目` | 仿宋 | 12pt (小四) | 正常 | 左对齐，首行缩进 |
| **引用** | `> 内容` | 仿宋 | 11pt | 斜体 | 左对齐 |

## 格式说明

### 一级标题（Level 1）
- **字体**: 方正小标宋简体
- **字号**: 22pt（二号）
- **样式**: 加粗
- **对齐**: 居中
- **间距**: 段前180pt，段后60pt
- **示例**:
  ```markdown
  # 一、 强化思想引领，筑牢信仰之基
  ```

### 二级标题（Level 2）
- **字体**: 楷体
- **字号**: 16pt（三号）
- **样式**: 加粗
- **对齐**: 左对齐
- **间距**: 段前120pt，段后40pt
- **示例**:
  ```markdown
  ## 二、 聚焦学业发展，强化学风建设
  ```

### 三级标题（Level 3）
- **字体**: 仿宋
- **字号**: 16pt（三号）
- **样式**: 加粗
- **对齐**: 左对齐
- **间距**: 段前100pt，段后30pt
- **示例**:
  ```markdown
  ### 三级标题内容
  ```

### 正文
- **字体**: 仿宋
- **字号**: 12pt（小四）
- **行距**: 1.5倍
- **对齐**: 左对齐
- **首行缩进**: 2字符（约0.3英寸）

## 特殊格式

### 加粗文本
使用双星号包围：
```markdown
**这是加粗文本**
```

### 斜体文本
使用单星号包围：
```markdown
*这是斜体文本*
```

### 列表
无序列表：
```markdown
* 列表项1
* 列表项2
```

有序列表：
```markdown
1. 第一项
2. 第二项
```

### 引用块
```markdown
> 这是引用内容
```

## 注意事项

1. **一级标题下方没有横线** - 已优化，标题下方不再显示下划线
2. **文件命名** - 第一个一级标题将作为输出文件名
3. **字体要求** - 确保系统已安装：
   - 方正小标宋简体
   - 楷体
   - 仿宋
4. **编码要求** - Markdown 文件必须使用 UTF-8 编码

## 公文格式标准

本工具遵循中国公文格式标准（GB/T 9704-2012）：
- 标题使用方正小标宋简体，二号
- 二级标题使用楷体，三号
- 正文使用仿宋，三号/小四
- 行距为1.5倍
