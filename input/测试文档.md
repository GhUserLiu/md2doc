在Word中，Markdown中以 `*` 和空格开头的格式通常对应：

## 对应关系

### 1. **无序列表（项目符号列表）**
- **Markdown:** `* 项目内容`
- **Word对应:** 项目符号列表（如•、◦等符号）

### 2. **实际示例对比**

| Markdown | Word中的效果 |
|---------|------------|
| `* 项目一` | • 项目一 |
| `* 项目二` | • 项目二 |
| `  * 子项目` |   ◦ 子项目（缩进二级） |

### 3. **其他类似情况**
- `+ 内容` 或 `- 内容` 也对应Word的项目符号列表
- 嵌套列表（缩进的 `*`）对应Word的多级项目符号

## 在Word中手动设置的方法：
1. **开始**选项卡 → **段落**组 → **项目符号**按钮
2. 选择喜欢的符号样式
3. 使用 **增加缩进**/**减少缩进**按钮调整层级

## 转换注意事项
当从Markdown导入到Word时（如使用Pandoc、Typora等工具），`*`开头的行会自动转换为项目符号列表格式，保持原有的层级结构。
